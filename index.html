Le projet des 40 layouts

Explications :

En 2005, Alessandro Fulciniti propose un HTML assez simple avec 40 variantes d’affichages pour gérer l’affichage d’une page web avec :

  1. entête
  2. pied de page
  3. contenu
  4. navigation principale
  5. navigation secondaire

15 ans après, ce code fonctionne toujours, mais mériterait d’être actualisé, je l'ai donc fait dans le cadre d'un stage chez pyrat.net avec ce que permettent les CSS modernes : flexbox et grid layout.

Le site à l'origine du projet est donc accessible

Objectif du projet :

Refaire les 40 layouts

  • en gardant le HTML tel quel
  • et en refaisant les CSS de manière moderne (grid et flexbox)
  • et en en profitant pour équilibrer la hauteur des colonnes
  • et faire les déclinaisons RWD pour tablettes et mobile
  • et de faire un site de démo semblable à l’original pour faire la démonstration des 40 layouts
  • le tout en open source, publié sur Github

L'open source

Il faut savoir que l'open Source est un code conçu pour être accessible au public : n'importe qui peut voir, modifier et distribuer le code à sa convenance.


Comme dit auparavant mon projet est totalement en open source, disponible sur github.


Vous pouvez donc retrouver le code entier ainsi que toutes les ressources
